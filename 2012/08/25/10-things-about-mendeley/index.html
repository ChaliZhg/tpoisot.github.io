Ten things about Mendeley

You may not know it, but I am a Mendeley advisor. I organize training sessions, and I answer questions by people wanting to use their software. It's quite fun, and I've learnt a few things in the process. I'd like to share 10 things about Mendeley; five I love, and five which fill me with unspeakable frustration. This is not an impartial review. By all means, if I'm biased, its towards Mendeley, and not the opposite.

The things I love

1. It integrates with LaTeX!

2. Sharing and collaborating is great

3. I can manage my own references

4. It sorts my PDF on its own

5. It works from within my browser

The things I hate

1. Cross-platform is a lie!

This is probably the one which annoyed me the most. I recently switched from Mac OS to Ubuntu. I still have a few manuscripts in .doc format, so I figured I will convert them for LibreOffice, and resume working. I dropped a note to the Mendeley support, and asked how to do it. The response was really clear: you can't. You can export from a format to the other only in some platforms. Fuck that. What is the point of using a markup format if you cannot write your plugin correctly? Where is the difficulty of parsing something which, in word, might read like `` into the equivalent format for OpenOffice? Bad design. The cross-platform ability works in the sense that you can use the same program on different platforms. You cannot actually collaborate with users using different platforms.

Let me put it another way. It's exactly like if you were unable to compile the bibliography of a LaTeX document using TeXWorks, because it was first written using Kile. This will never happen, because when you write a citation as \cite{citation_key}, then you can parse this with any software you want. That Mendeley is unable to convert it show that they do not care a lot about inter-operability of the documents. It's just bad news for the users. It can be solved by having something like EndNote's Scan RTF function. If EndNote can do something your bibliography app can't... you have some coding to do.

2. The mobile application is a (bad) joke

I'm not always near my laptop, and sometimes I just want to check something without booting it up. Mendeley has a free iOS/Android application available for just this situation. It's called Mendeley Lite, but sadly Lite does not even begins to describe it.

4. The search function is awful

A few final words

Obviously, the good things, by far, outnumber the bad. Mendeley is worth trying, and worth using, because it will save you a lot of time in the long run. Having the ability to work with LaTeX and Word-like software is good.
