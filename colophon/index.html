Technologies

This website is published on GitHub pages, and generated using jekyll and pandoc. The css files are generated using less. Most likely, the content was written using Atom on a laptop running Arch Linux. The fonts used (Roboto and Iconsolata) are provided by the Google Fonts project. Equations are rendered using MathJaX, and d3.js may be used for a few graphs here and there. Other libraries used are skeleton, fontawesome, and highlight.
